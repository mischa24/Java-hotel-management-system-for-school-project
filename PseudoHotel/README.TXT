Περιγραφή Project – Hotel Booking Management App (Java)
✅ Γενική Ιδέα
Φτιάξαμε μια πλήρη εφαρμογή σε Java για τη διαχείριση κρατήσεων σε ένα μικρό ξενοδοχείο.
Η εφαρμογή επιτρέπει την καταχώριση, αποθήκευση και επεξεργασία κρατήσεων, υπολογίζει εκπτώσεις, ΦΠΑ και εμφανίζει αναλυτικές αποδείξεις για κάθε πελάτη.

✅ Σενάριο Χρήσης
✔️ Ο χρήστης (π.χ. ο υπάλληλος υποδοχής) ανοίγει το πρόγραμμα.
✔️ Ορίζει πόσες κρατήσεις θέλει να εισάγει.
✔️ Για κάθε κράτηση εισάγει τα πλήρη στοιχεία πελάτη και δωματίου.
✔️ Το πρόγραμμα υπολογίζει αυτόματα υποσύνολο, έκπτωση, ΦΠΑ και τελικό ποσό.
✔️ Εμφανίζει πλήρη αναλυτική απόδειξη.
✔️ Στο τέλος δείχνει στατιστικά:

✔️ Υποσύνολο = (διανυκτερεύσεις * τιμή ανά διανυκτέρευση)
✔️ Έκπτωση:

Αν διανυκτερεύσεις > 5 → έκπτωση 10%

Extra έκπτωση 5% αν ποσό > 100€
✔️ ΦΠΑ = 13% στο ποσό μετά την έκπτωση
✔️ Τελικό Ποσό = Έκπτωση + ΦΠΑ
